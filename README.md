# ChatApp Vue.Js

## Start Project 

First open new terminal <br> 
1-) <code> cd chat-server </code> <br>
2-) <code> node server.js </code> <br>

First open new terminal <br> 

1-) <code> cd chat-app </code> <br>
2-) <code> npm run dev </code> <br>
